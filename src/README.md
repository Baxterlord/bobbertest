# Quellcode des Projekts

## Struktur des Codes

`src/`                  - Enthält den Quellcode des Projekts.

   `/javascript/`       - Enthält alle JavaScript-Dateien
   `/html/`             - Enthält alle HTML-Dateien
   `/java/`             - Enthält alle Java-Klassen
   `/sql/`              - Enthält alle SQL-Dateien
